# 📊 Student Placement Data Analysis

## 📌 Project Description
This project analyzes student placement data using Python.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📁 Project Structure
placement-analysis/
│
├── data/
│   └── placement_data.csv
│
├── src/
│   └── analysis.py
│
├── outputs/
│   └── (generated charts)
│
├── requirements.txt
├── README.md

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python src/analysis.py

## 📈 Features
- Placement status analysis
- CGPA vs Salary visualization
- Branch-wise placement comparison
- Correlation heatmap

## 📊 Insights
- Higher CGPA increases placement chances
- Internships and projects improve job opportunities
